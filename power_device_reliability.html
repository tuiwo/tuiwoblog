复杂的时间 Part 1

2026-02-13 | 功率器件,可靠性,蠕变

我本来想先写疲劳相关的内容，但还是决定直接聚焦到最初困扰我的那些问题上：

  1. 器件工作时，不断延长循环时间怎么影响器件寿命？
  2. 时间是怎么产生破坏并影响器件寿命的？

这些问题都与曾经困扰我的那个现象相关：在产品验证环节，一颗通过相关标准验证的芯片（在数分钟热循环工况中通过1000次测试）轻松通过了以天为周期的带载循环（数十次），但是在小时级别的带载循环中键合线寿命只有200次左右。

从三个实验数据中任意取两个点计算时间的影响，可以发现存在两个截然不同的区域：

  1. 分钟级-小时级：循环寿命迅速降低；
  2. 小时级-天级：循环寿命变化相对较小。

尽管在这个案例中键合线寿命的表现是惊人的（很大程度上可能要归因于该器件质量不达标），但这个现象实际并非偶然，只是此处不合适的设计（内部结构、材料等）放大了这个问题。例如，我们很快可以在infineon的应用手册中找到相关的结果：功率器件的功率循环能力在0.1s-60s的范围内随加热时间延长而下降，在加热时间大于60s的范围内则近似维持稳定1。

关键是为什么？

等温热循环与饱和现象

通过前面对蠕变的介绍，我们容易产生一种错误的印象：既然在恒定应力的情况下，蠕变导致的形变会随着时间不断积累，那么无限延长循环时间将会导致一个热循环就可以摧毁连接处。

但实际中热循环过程如下（此处以最经典的元件-焊点-衬底结构为例，芯片的热膨胀系数\(\alpha _{c}\)，底板热膨胀系数 \(\alpha _{s}\)，且存在关系 \(\alpha _{c}<\alpha _{s}\)，结构中所有部分温度是相同的，同样的结论也适用于其他焊面）：

Figure 1: 热循环中焊接结构的变形

可以发现，当整体结构处于高温状态时，由于上面的组件和下面的底板由于热膨胀系数不匹配，其横向变化量不同，底板受热伸长更多，相当于将焊料横向拉长；处于低温状态时，底板收缩更多，同样将焊料挤压。

热循环最高温处，焊料水平方向形变：

\begin{equation} L_{max} = L(\alpha _{s} - \alpha _{c} ) (T_{max}-T_{0}) \end{equation}

热循环最低温处，焊料水平方向形变：

\begin{equation} L_{min}= L(\alpha _{s} - \alpha _{c} ) (T_{min}-T_{0}) \end{equation}

换句话说，热循环中连接结构经历的并不是恒定应力过程，而是恒定应变过程：每次系统处于高温点时，焊料都被上下结构拉伸到同一个位置；每次系统处于低温点时，焊料都被上下结构挤压到同一个位置。

热循环这种特性决定了焊接结构的蠕变并不像蠕变试验中那样受恒定应力无限发展，而是存在边界。显然，在一次热循环过程中，焊料发生的形变不会超过 \(L_{max}-L_{min}=L(\alpha _{s}-\alpha _{c})(T_{max}-T_{min})\)，那么一次热循环中所能累积的最大非弹性形变也不会超过这个数值，而真正威胁器件寿命的就是这些非弹性形变。换句话说，即便无限延长温度循环中高温阶段的时间，焊料并不会因无限增长的蠕变形变发生失效，因为该过程中总的非弹性形变会不断增长但被约束在上述边界值之内，存在饱和现象。

另外一个与之对称的问题是：如果无限缩短加热时间，材料的形变和寿命如何变化？这里我们可以简单地思考其中的逻辑关系：工程材料的三种主要变形形式（弹性变形、塑性变形、蠕变变形）中只有蠕变变形是与时间相关的，弹性变形和塑性变形都是瞬时发生的。尽管塑性变形与蠕变变形的物理机理类似，但是塑性变形在材料受力超过一定限度时就瞬间发生。

当无限缩短循环时间时，蠕变变形的影响消失了，只需要考虑其余两者。换言之，在某个固定的温度条件下进行循环，无限短的循环时间使材料处于循环应力或者循环形变的情况下。同样，只要温度循环的最高温和最低温保持不变，材料承受的应力或者应变就不变，此时结构的寿命同样发生饱和。

综合来看我们可以总结如下：只改变温度循环的循环时间、其他条件保持不变的情况下，随着循环时间不断延长，焊接结构的寿命不断降低但最终饱和到一个固定的下界；同样随着循环时间无限缩短，焊接结构的寿命不断增加但最终会饱和到一个固定的上界。

蠕变变形模型与松弛行为

为了直观地展示影响焊接结构寿命的、与时间有关的过程，首先利用最简单的模型进行一些数学描述，这个模型使用一个弹簧模拟材料中的弹性变形，使用一个黏性环节模拟材料中与时间有关的变形，如下图所示。

Figure 2: 蠕变的最简模型

恒定应力情况

简化模型中，弹簧在外力的作用下发生形变 \(\varepsilon_{e}=\sigma/E\)，黏性环节通过一阶导数引入时间的作用: \(d\varepsilon_{c} / dt = \sigma/\eta\)(\(\eta\)为黏性环节的黏性系数，为一个常数)，因此总形变：

\begin{equation} \varepsilon = \varepsilon_{e} + \varepsilon_{c} = \frac{\sigma}{E} + \frac{\sigma}{\eta}t \end{equation}

随着时间延长，在恒定应力的作用下材料的变形无限增长，因此我们要把注意力转向更符合实际的情况。

恒定应变情况（松弛行为）

假设模型的总形变突然被拉到 \(\varepsilon_{0}\)的位置并保持不变，在初始瞬间，黏性的蠕变行为无法提供任何形变支持，全部的形变都由弹簧提供，即初始条件: \(\varepsilon_{c}(0) = 0, \varepsilon_{e} (0)= \varepsilon_{0} , \sigma(0)= E \varepsilon_{0} \)

任意时刻都有：

\begin{equation} \varepsilon_{0} = \varepsilon_{e}(t) + \varepsilon_{c}(t) \end{equation}

上式对时间进行求导可以化为

\begin{equation} \frac{\dot{\sigma}(t)}{E} + \frac{\sigma(t)}{\eta} =0 \end{equation}

这个简单的方程的解是

\begin{equation} \sigma(t) = E \varepsilon_{0} e^{ - \frac{E}{\eta} t } \end{equation}

对应的弹性变形和蠕变变形结果分别是

\begin{equation} \varepsilon_{e}(t)= \varepsilon_{0} e ^{- \frac{E}{\eta}t} \end{equation} \begin{equation} \varepsilon_{c} (t) = \varepsilon_{0} (1- e^{-\frac{E}{\eta}t}) \end{equation}

不难发现，在恒定应变的情况下，起初全部的形变由弹性变形提供，随着时间推移，累积的蠕变变形增长，由于总形变不变，因此原本的弹性变形被渐渐取代，对应的，系统中的应力也逐渐减轻，这个过程也称为应力松弛。

应力松弛为器件正常工作中累积的非弹性变形提供了解释思路：尽管器件正常工作时所受的应力并不大，在此应力下器件的应变应该全都是弹性的，但是只要给予一定的温度和时间，蠕变就能缓慢发生、累积，其导致的非弹性形变将逐步替代原本的弹性形变。

我们可以看到，如果撤去外力或者施加的应变，上述简化模型中积累的蠕变应变将不会自发恢复、消失。尽管有些蠕变形变实际上是可以自发恢复的（例如在高分子材料中），但是在金属和陶瓷材料在所处温度超过50%熔点时发生的较大的蠕变应变只能产生很小的恢复，因此这个简化模型对我们来说是够用的。

还有一些名词可能与上述概念相关，但看上去它们穿了不太一样的衣服：

  1. 瞬态蠕变：在第一阶段中的蠕变变形（见图1），但理想情况下有时也认为是可恢复形变；
  2. 稳态蠕变：在第二阶段中的蠕变变形（见图1），但是有时说的也是不可恢复的形变；
  3. 黏弹性：指的是受力发生与时间有关的变形，卸载后变形可以自动恢复，例如上面所说的高分子材料中的蠕变；
  4. 黏塑性(viscoplastic)：指的是受力发生与时间有关的变形，变形包括不可恢复的永久塑性组成，例如上面所说的金属中的蠕变。

简单地了解这些名词的概念有助于阅读一些技术手册，例如工程本构模型或数值分析中，有时将蠕变与黏塑性变形等同（即视为一种率相关塑性行为）2，现在我们就不难理解脚注1中表达的含义了。

  • 弹性环节的影响

    在封装的焊接结构中，弹性有两部分来源：

    1. 焊料自身的弹性
    2. 上下结构提供的弹性：例如IC封装的引脚，键合线结构

    相对而言，焊料自身的弹性应该是非常小的。

    较大的弹性有益于短期内减轻蠕变所致的危害，这个过程可以通过公式(21)说明。如果给封装结构施加10%的恒定应变并保持不变，弹性不同的封装的蠕变应变与时间的关系如下图所示：弹性较好的结构有很小的E，其蠕变累积的速度更慢，也需要更长的时间才能将全部的应变以蠕变变形取代；弹性较差的结构有很大的E，其蠕变积累的速度很快，在很短的时间内10%的总应变就全部转化为蠕变变形。

    由于处于恒定应变情况，充分长的时间以后，蠕变形变所能积累的极限也是10%，因此最终所有结构将会达到相同的蠕变应变值。

    在恒定应变工况下，较好的弹性结构能够在轻易提供大量的弹性形变的同时确保内部应力较低(\(\sigma=E \varepsilon\)，其中\(E\)较小)，因此蠕变发生的速率相对较慢(\( \dot{ \varepsilon_{c} }= \sigma/\eta \))，因此在饱和前的相同时间内，弹性更好的结构蠕变形变更小。

    Figure 3: 不同弹性对蠕变过程的影响

  • 黏性系数的含义

    如果使用黏性环节模拟蠕变现象，结构的黏性系数 \(\eta\) 应该有清晰的物理意义。显然

    \begin{equation} \eta = \frac{\sigma} { \dot{\varepsilon_{c} }} \end{equation}

    根据上一篇中的公式(8)可以写出 \(\eta\) 的复杂表达。

    实际中最简单也最受关注的情况是稳态蠕变阶段，此时蠕变速率达到稳定值，不再随时间变化，这里我们使用公式(9)，可以得到：

    \begin{equation} \eta = \frac{1}{A\sigma^{n-1}} \exp{ \left(\frac{Q}{RT} \right) } \end{equation}

    可以看到，即便不考虑时间和温度的影响，在稳态、恒温的近似情况下，得到的黏性系数仍然不是一个稳定的常数。在有些机理的驱动下，n可能取大于1的值，此时随应力增加蠕变速率剧烈增长，得到的黏性系数也会发生变化。

    在应力松弛的现象中，结构内部的应力不断变化，结合依赖于应力的黏性系数，可以发现我们需要求解一个非线性响应而不是原本的方程(18)，此时对(17)求导得到（恒温条件下不再写出温度项，即合并到\(A\)中）：

    \begin{equation} \frac{\dot{\sigma}(t)}{E} + A [\sigma(t)]^{n} =0 \end{equation}

    无须求解也可以直观感觉到：在应变刚施加的开始时刻附近，蠕变积累的速度相当惊人，因为此时蠕变应变与所受的应力呈现指数倍关系，而在初试时刻结构的应力是最大的。

    在\(n>1\)的情况下，上式的解析解为（\(\sigma_{0}\)是初始应力）

    \begin{equation} \sigma(t) = \frac{\sigma_{0}}{[{t A E(n-1) {\sigma_{0}}^{n-1} +1}]^{1/(n-1)}} \end{equation}

恒温恒应变工况

针对温度循环的分析显示本质上这是一种恒应变老化工况，同时，根据简单的模型我们现在也可以描述蠕变现象随时间、应力的变化，只是没有考虑温度的影响。

这些信息启发我们，可以在恒温条件下周期性地给焊料施加恒定应变从而模拟温度循环中连接处的老化过程，这个过程如下图所示。

Figure 4: 恒温条件下周期性恒定应变工况

在外部施加的应变的作用下，焊料会发生形变，并且会积累一些蠕变变形，这些变形使得焊料的长度偏离原本的自然位置。经过一段时间以后，系统会到达稳态：每个正应变半周期的末尾(\(t_{1}\))，焊料在\(x\)方向上的累积的非弹性形变都正好累积到固定的位置，这里假设为\(x_{p}\)；然后施加的应变反向，在负应变周期的末尾(\(t_{3}\))，焊料在该方向上又恰好被压缩到同一个位置，也就是\(-x_{n}\)。

假定应变转换的过程很快(\(t_{1}-t_{2},t_{3}-t_{4}\))，在此期间发生的蠕变形变就可以忽略，也就是说\(t_{2}\)时虽然总应变已经被反向压缩，但是焊料本身长度依然位于\(x_{p}\)(与\(t_{1}\)时刻相同)，其余的部分\(- \frac{\Delta x}{2}\to x_{p}\)都有瞬时的弹性形变提供。

由于正半周期和负半周期条件都是一致的，没有理由认为两者之间有差别，根据这种对称性应该有 \(x_{p}=x_{n}\)。

因此此处只分析\(t_{2}\to t_{3}\)的过程。为了方便计算，选择反方向(图中\(x'\)方向)作为正方向，并且选择\(t'\)作为时间变量，\(t'\)以\(t_{2}\)时刻作为起始点。

类似于(17)-(21)的分析，可以列出：

\begin{equation} \varepsilon_{e}(t') + \varepsilon_{c}(t') = \frac{\Delta x}{2} \end{equation}

求导。为了观察非线性条件下的过程，根据对黏性系数的认识，此处选择(24)中的蠕变速率定义，

\begin{equation} \frac{\dot{\sigma}(t')}{E} + A [\sigma(t')]^{n} =0 \end{equation}

初始条件如下：

\begin{equation} \sigma(t'=0) = \left( \frac{\Delta x}{2} + {x_{p}} \right)E \end{equation} \begin{equation} \varepsilon _{c}(t'=0) = -x_{p} \end{equation}

(27)解的形式其实就是(25)，为了方便计算，选取\(A=1,n=2\)分析，也就是说 \(d {\varepsilon_{c}}/dt = \sigma^{2}\)，于是解可以写成

\begin{equation} \sigma(t') = \frac{\sigma (t'=0)}{E \sigma(t'=0)t+1} \end{equation}

我们只关心焊料中的非弹性形变（蠕变形变）。使用约束(26)，此时弹性形变可以直接写出，因此就可以简单求出任意时刻的蠕变形变

\begin{equation} \varepsilon_{c} (t') = \frac{\Delta x}{2}- \varepsilon_{e} (t') = \frac{\Delta x}{2}- \frac{\sigma(t')}{E} \end{equation}

经过了半个周期(\(T_{s}\))以后到\(t_{3}\)时刻时，蠕变变形恰好累积到\(x_n\)的位置，即

\begin{equation} \varepsilon_{c} (T_{s})= x_{n} = x_{p} = \frac{\Delta x}{2}- \frac{\sigma(T_{s})}{E} = \frac{\Delta x}{2}- \frac{\sigma (t'=0)}{E^{2}\sigma(t'=0)t+1} \end{equation}

实际中，我们关心的是蠕变应变的应变振幅，也就是\(x_{p}+x_{n} =2x_{p}\)，将上面的式子同时乘以2，可以得到：

\begin{equation} 2 x_{p }= \Delta x - \frac{ ( \Delta x +2x_{p} )}{E^{2 } ( \frac{\Delta x}{2} +x_{p} )T_{s} +1} \end{equation}

设振幅为 \(\Delta \varepsilon =2x_{p}\)，则

\begin{equation} \Delta \varepsilon_{}= \Delta x - \frac{ ( \Delta x + \Delta \varepsilon)}{0.5 E^{2 } ( \Delta x + \Delta \varepsilon )T_{s} +1} \end{equation}

取\(\Delta x =10\%\)，即恒定应变为参考长度的百分比数值；\(E\)根据材料弹性选取，为了观察其影响，我们选择0.01, 1, 100, 10000观察其对振幅的影响。将(34)转化为\(\Delta \varepsilon\)与\(T_{s}\)的二次函数式，可以得到两者的关系，其余变量均已经给定，绘制图像如下。

Figure 5: 蠕变振幅与循环时间的关系

图9与图7十分相似，即便我们采用了非线性的模型来分析蠕变过程，但是结论依然可以总结如下：

  1. 焊接结构在恒温条件下经历恒应变作用时（该工况与温度循环十分相似，只是没有温度变化），蠕变导致的非弹性形变振幅随着循环周期延长增加，但最终将达到饱和值；
  2. 因此，对于特定的材料，此工况下其寿命在一定时间尺度内（较短的时间尺度）降低，到达另一时间范围时寿命缩短变慢，最终寿命将会趋近于一个下界；
  3. 对于上述较短的较短时间尺度的循环工况，增加材料的弹性可以显著影响其蠕变应变的振幅，延长寿命；
  4. 但是如果循环的周期很长（或者说频率很低），改变材料和结构的弹性对缩短应变振幅没有帮助，因此不会延长寿命。

在施加的条件下，我们假定存在一种周期性反复拉压的非弹性形变模式，通过求解前面建立的解析模型（通常这个求解是通过仿真完成的），我们确实找到了这样的解。

当结构在外力的作用下发生反复的、周期性的非弹性形变过程时，即便在单方向上都没有发生无限的非弹性形变（例如恒定应变约束），材料内部周期性的、有限的非弹性形变依然会造成失效，这个过程就是疲劳。

看上去，由于温度循环和上述恒应变工况的相似性，焊接结构在温度循环中的失效似乎可以完全归因于这种周期性蠕变变形所引发的疲劳过程，并结合图9与图7的结果解释循环寿命与时间之间复杂的依赖关系。

但事实并非如此，温度条件的缺失使得整个故事变得乐观：我们一直置之不理的温度其实会显著影响这个过程，我们将会结合文献中经典的结构再来完整地审视这个过程，并且看到几何结构和蠕变棘轮是如何作用的。

简介

实际中，硬件周期性地加热和冷却会产生应力3，这些应力引将疲劳(fatigue)和蠕变(creep)耦合在一起，影响了器件封装中关键结构的寿命。对功率器件封装而言，以下几个焊接界面是最关键的结构：芯片表面的键合线结构、芯片底面的焊料结构、DBC与基板(baseplate)的焊料结构。

对我而言，蠕变这种时间相关的机制很奇特，因此我将从此开始。蠕变令人深刻的一点是它引发的变形会随时间推移而积累，而通常我们理想化地认为弹性应变和塑性应变是应力的瞬时响应。时间作为变量引入将会让分析顿时变得复杂。

不同材料对蠕变的敏感度不同。通常金属材料在处于绝对熔点温度的30%~60%时才会表现出显著的蠕变变形；外界条件相同时，铬镍铁合金的蠕变变形发生速度明显慢于不锈钢、铝合金等材料，因此高温环境中使用的金属材料通常都包含铬镍钴等合金化元素。

蠕变试验现象

典型的蠕变测试是在恒温、恒定外力情况下测量样品随时间变化的变形，如下图1所示。在此过程中，最重要的特性是应变积累的速率 \(\dot{\varepsilon} = \frac{d \varepsilon}{dt}\).

  • 施加负载以后，发生瞬时弹性应变（也许还有塑性应变），然后蠕变逐渐累积；
  • 蠕变第一阶段(初级蠕变/瞬时蠕变/primary creep)：此阶段内，\(\dot{\varepsilon}\)初始较大然后不断减小，最终接近常数；
  • 蠕变第二阶段(稳态蠕变/steady-state creep/secondary creep)：蠕变速率 \(\dot{\varepsilon}\) 是恒定的，即曲线是线性的。在该蠕变阶段通常持续的时间最长；
  • 蠕变第三阶段：速率会增加并导致最终的失效，这种失效通常称为断裂。

Figure 6: 恒定载荷下蠕变随时间的发展

从蠕变曲线中，我们可以得到2条直觉认识：

  1. 虽然外力/负载恒定，但材料的变形并不是稳定的，而是随着时间推移不断累积；
  2. 这种时变的蠕变变形在不同的时间尺度上特性也是不同的，在不同的阶段其蠕变速率不同。

蠕变的物理机制

我们应该稍微了解一下金属、合金这样的晶体材料蠕变的物理机制。蠕变涉及的复杂物理机制总体上可以分为两大类：扩散流动和位错蠕变。

扩散流动

  • 晶体中的空位缺陷
    • 固体材料可以根据原子或离子间排列的功率进行分类：
      • 晶体：原子在很大范围内重复或周期性排列；
      • 非晶：原子的排列缺乏大范围的排列规律
    • 单晶 多晶 晶界
      • 单晶：原子的周期性和重复性排列非常完美，延伸到整个样品而没有中断；
      • 多晶：由很多小的晶体或晶粒(grains)组成；
        • 晶界(grain boundary)：不同晶粒的交界面
    • 实际晶体中存在着大量缺陷(点缺陷、线缺陷、面缺陷)
      • 空位(vacancy)：正常的节点位置缺少了一个原子
      • 自间隙原子：一个原子挤进了间隙位置

    Figure 7: 晶体中的空位缺陷

  • 空位扩散

    当晶体承受应力时，空位更容易在与应力方向垂直的晶界附近自发形成，导致此处空位浓度较高，从而引发空位从高浓度区域向低浓度区域扩散，最终导致晶粒形状发生变化，引发蠕变变形。

位错蠕变

  • 晶体中的线缺陷：位错

    位错指的是晶体内存在线状错排，包括刃型位错和螺型位错。我们在这里只展示刃型位错来获得一些简单直观的理解。

    刃型位错如下图所示，这种缺陷的特征是有一个原子平面在晶体内部的一条边缘线(位错线)上终止了。在位错线周围原子排列会产生局部畸变。

    Figure 8: 刃型位错缺陷

  • 位错移动和塑性变形

    塑性变形和大量的位错运动有关。当晶体承受 切应力 的时候，位错会发生滑移(slip)，每次只移动一个原子间距，最终这个终止的面会移动到晶体的表面，形成一个台阶，这个过程如下图所示。位错滑移所在的平面称为滑移面。

    大量位错滑移面集中处最终会产生大量的位错滑移台阶，塑性变形经常集中于此。

    Figure 9: 位错滑移与滑移台阶

  • 位错蠕变(也称为幂律蠕变)

    一种用来解释蠕变的位错移动机制还位错攀移。当位错的滑移面上存在障碍时，位错无法继续滑移；此时如果想要继续变形位错就只能离开原本的滑移面，迁移到相邻的平面，这个过程称为攀移。这种位错攀移需要结合空位缺陷扩散的方式，因此这种蠕变的过程似乎是受到扩散控制的。此外，这种攀移过程也是和时间相关的。

机制总结 扩散 Arrhenius方程

通过对蠕变物理机制简单的了解，可以发现蠕变与扩散过程(两种机理都涉及空位的扩散)相关。

扩散指的是通过原子运动而形成的物质输运现象，这个过程发生要求满足2个基本前提：

  1. 有空缺的相邻位点，即有可以转移的位置；
  2. 原子得有足够的能量解除原本位置的束缚并完成移动，即有能量
  • 扩散发生有多快：传质速率(Rate of mass transfer)

    通常用扩散通量(diffusion flux, \(J\))衡量扩散发生的速率，即传质速率。

    扩散通量 \(J\)定义为为单位时间内垂直通过单位截面积上的扩散物质质量\(M\)(或原子数：

    \begin{equation} J= \frac{M}{At} \end{equation}

    其中， \(A\)表示扩散发生的截面积， \(t\) 表示扩散时间。

    很容易拿电流来进行简单的类比，如果输运的物质恰好是电荷 \(Q\)，那么简单来看：

    • 电流的定义 \(I=Q/t\)
    • 电流密度的定义 \(J=I/A=Q/At\)

    显然扩散通量之于物质输运，就如同电流密度之于电荷移动：当电流密度很大的时候，说明电子正在进行快速运动。

  • 扩散系数(diffusion coefficient)

    使用电流来类比物质输运是非常自然的，在半导体物理里面是顺理成章的推导，在这里我们将也能获得一些直觉结果。

    在稳态情况（即任意点上的状态不会随时间发生变化）下，如果材料内部物质浓度 \(C\) 存在不均匀，那么在浓度差异的驱动下将会产生扩散，这个过程可以通过Fick's first law进行描述：

    \begin{equation} J=-D \frac{d C}{d x} \end{equation}

    其中，\(D\)称为扩散系数；\(\frac{dC}{dx}\)表示沿着\(x\)方向的浓度梯度。

    假设扩散在单位面积的材料内相差单位距离的两点(m,n)之间发生(\(A=1,dx=1\))，上式就是

    \begin{equation} J=-DdC \end{equation}

    再使用欧姆定律进行简单类比，此时 \(A=1\)，所以电流密度就等于电流，假设m点和n点之间的电压（电势差）为\(V\)，电阻为\(R\)，电导为\(G\)，那么：

    \begin{equation} I = \frac{1}{R}V = GV \end{equation}
    • 驱动力：电压-浓度差
    • 速率：电流密度-扩散通量
    • 系数：电导(电阻倒数)-扩散系数

    因此，扩散系数\(D\)的大小描述了原子扩散承受的阻力，或者说扩散的难易程度：

    • \(D\)较大意味着扩散的“电导”比较大，或者说“电阻”比较小，扩散很容易进行；
    • \(D\)较小意味着扩散的“电导比较小”，或者说“电阻”比较大，扩散较难进行。
  • Arrhenius关系

    前面提到过，扩散发生的必然要求之一是有能量，即原子能量水平影响了扩散的难易程度，现在我们可以建立能量和扩散系数之间的关系：

    • 当原子能量比较高的时候，扩散很容易发生，因此扩散系数\(D\)比较大；
    • 当原子能量比较高的时候，扩散难以发生，因此扩散系数\(D\)比较小。

    影响原子能量的重要因素是温度：

    • 温度提升会加剧原子相对于自身平衡位置的平均振动，在此能量水平下，原子更容易离开原本的位置发生扩散
    • 这种情况就是热激活的例子。

    这个温度-能量-扩散难易程度-扩散系数的关系可以通过Arrhenius关系来描述，即扩散系数存在温度相关性：

    \begin{equation} D=D_0 \exp \left( - \frac{Q_d}{RT} \right) \end{equation}

    其中，\(D_0\)是个常数，\(Q_d\)是扩散所需要的激活能，\(R\)是气体常数，\(T\)为绝对温度。

    在热激活的物理过程中，Arrhenius关系的应用很广泛；其中激活能可以单位物质发生热激活过程（例如扩散）所需要的能量:

    • 所需要的激活能越大，说明该物理过程越难发生，在扩散过程中，对应扩散系数越小。

    从物理机理上，蠕变过程受扩散过程控制，因此可以预期蠕变的应变发生速率与温度存在一个Arrhenius项，即：

    \begin{equation} \dot{\varepsilon} \propto \exp \left( - \frac{Q_d}{RT} \right) \end{equation}

    所以接下来有必要从蠕变物理机理涉及的因素出发刻画蠕变变性的数学形式。

蠕变模型

蠕变的物理机制揭示了几个重要的影响因素：

  1. 时间：从蠕变试验的曲线上可以清晰看到这种复杂行为；此外，位错攀移和时间有关；蠕变引发的塑性形变显然也会随着时间积累；
  2. 应力：例如当承受切应力的时候，会引发位错滑移；
  3. 温度：通过提供更多能量加剧热激活过程（扩散）。

这些因素导致了一些重要的现象：

  1. 金属和陶瓷：发生蠕变通常意味着较高的应力水平（接近屈服点），温度约为50%熔点之上；
  2. 电子焊料(60Sn-40Pb)：熔点仅为185摄氏度，因此在室温条件下也具有蠕变潜力，\(298K/458K=0.65\)；
  3. 钨丝要在非常高的温度里工作，因此需要强化抗蠕变能力；
  4. 电子封装中的陶瓷基板熔点在2000摄氏度以上，因此常规使用不会蠕变。

模型表达式

时间 \(t\)、温度 \(T\)、应力 \(\sigma\)对蠕变应变 \(\varepsilon\)的影响可以通过下面的表达式描述：

\begin{equation} \varepsilon = f(\sigma)g(t)\exp \left( - \frac{Q}{RT} \right) \end{equation}

其中 \(Q\)表示蠕变激活能， \(f(\sigma)\)表达式描述应力的影响，有以下不同的形式：

Name \(f(\sigma)\)
Norton \(A \sigma^n\)
Prandtl \(B\sinh(b \sigma)\)
Dorn \(C\exp(c \sigma)\)
Garafalo \(D\sinh(f \sigma)^n\)

在不同的蠕变阶段（瞬态蠕变、稳态蠕变、第三阶段蠕变）蠕变行为显然也不一样，这种特性通过 \(g(t)\)刻画：

Stage \(g(t)\)
primary creep \(1+et^{1/3} \)
secondary creep \(t\)
tertiary creep \(t^m\)

进一步，应变速率可以写为：

\begin{equation} \dot{\varepsilon} = f(\sigma) \frac{d g(t)}{dt} \exp \left( - \frac{Q}{RT} \right) \end{equation}

稳态蠕变模型

第二阶段蠕变有较好的特性：

  1. 持续时间比较长；
  2. 蠕变应变率\(\dot{\varepsilon}\)是恒定的；

因此通常在这个阶段对易蠕变的材料进行条件设计。

稳态蠕变应变速率常用下面的现象方程描述：

\begin{equation} \dot{ \varepsilon_{sc} } = A \sigma^{n} \exp {\left( - \frac{Q}{RT} \right)} \end{equation}

更加详细的模型表示为：

\begin{equation} \dot{ \varepsilon_{sc} }= \frac{A'}{d^{q} T}\sigma^{n} \exp \left( - \frac{Q}{RT} \right) \end{equation}

其中， \(A', q, n \)参数的具体值取决于具体的蠕变机理， \(d\)是平均晶粒直径。

物理机制 \(n\) \(q\) 描述
扩散流动(Nabarro-Herring/Coble蠕变) 1 2 在低应力、高温度下发生
扩散辅助的位错蠕变 3~8 0 在高应力、中等温度发生

基于蠕变发生时内部物理机理的复杂性（如上表展示），可以预期公式(10)无法覆盖所有应力水平和温度下的蠕变变形。

  • 常用的焊料稳态蠕变模型

    例如在焊料的蠕变模型中，往往需要求助比公式(10)更复杂的形式：

    \begin{equation} \dot{\varepsilon_{sc} } = C \left( \frac{G}{T} \right)\sinh \left( \frac{a \sigma}{G} \right)^{n} \exp \left( - \frac{Q}{RT} \right) \end{equation}

    其中， \(C,a,n\)是常数， \(G\)是剪切模量。

蠕变的寿命估算：时间-温度参数

针对单纯的、恒定应力下的蠕变现象我们最后的一个问题是应该如何评估其寿命。

简单设想，如果我们能够将失效时间与影响因素的关系表示出来，就能通过控制变量法获得模型参数、唯一地计算失效时间。

虽然前面的分析缺乏直接可用的模型，但却可以导出性能指标(这里是蠕变应变)-时间关系。当处外力为\(\sigma\)、温度为 \(T\)时，稳态蠕变阶段的模型，例如公式(9)，显示应变速率 \(\dot{\varepsilon_{sc} }\)是个常数，据此粗略地估计时刻 \(t\)时蠕变应变为：

\begin{equation} \varepsilon = A \sigma^{n} t \exp \left( - \frac{Q}{RT} \right) \end{equation}

当应变累积到一定程度\(\varepsilon_{0}\)(常数)的时候认为发生失效，对应的时间可以视为寿命\(t_r\)，那么

\begin{equation} \varepsilon_{0} = A \sigma^{n} t_{r} \exp \left( - \frac{Q}{RT} \right) \end{equation}

接下来似乎只要利用控制变量的方法（加速实验）然后简单外推就能得到这个模型:

  • 首先固定应力\(\sigma\)，改变温度\(T\)，记录\(t_r\)，可以计算出激活能\(Q\)；
  • 其次固定温度\(T\)，改变应力\(\sigma\)，记录 \(t_r\)，可以计算指数 \(n\).

遗憾的是这样的方法似乎并不现实，当不同蠕变机理主导的时候，模型中的参数也会发生变化，这一点可以从上张表格的 \(n,q\)的取值中看出来：

  1. 激活能\(Q\)在温度和应力不同的情况下可能会因蠕变机理改变而变化，例如高纯铝在大约绝对熔点的50%以下时激活能减小；
  2. 当降低应力 \(\sigma\)获取 \(n\)时，主导机理的改变可能会导致 \(n\)在较小应力下发生突然改变，从而无法使用大应力实验得到的参数；然而，在这个小应力范围（但更接近实际使用情况）内，想要通过充分的实验得到真实的 \(n\)的取值不实际，实验可能持续几十年。

上述的外推方法在实际中效果不太好，为此需要引入一些观察、假设形成一些相对折中但也更成功方法，例如Sherby-Dorn参数法和Larson-Miller参数法，这些方法都涉及到时间-温度参数的使用。

Sherby-Dorn参数法

  • Sherby-Dorn参数法的关键假设是激活能\(Q\)是个常数；
  • 在此基础上，我们可以很简单地改变温度得到激活能，然后只需要解决\(n\)的问题就可以了；
  • 如果固定任意温度，然后降低应力以获得 \(n\)，那么随着应力水平逐渐降低，失效时间\(t_r\)大幅延长以致不可能完成；
  • 因此如果能够缩短低应力水平下的失效时间，就有希望在实验室内、在较短的时间内收集到更多可用的数据；
  • 观察方程(13)，当应力固定在一个低水平情况下， \(\theta_r = t_r \exp ( - Q/RT )=const\)，即\(\theta _r\)是一个常数，因此通过提高温度可以降低失效时间 \(t_r\)；
  • 因此，一个聪明的做法是：在降低应力水平的同时，提高蠕变的温度，从而将失效时间控制到可以进行的水平内；
  • \(\theta _r\)是一个时间-温度参数，其对数形式\(log \theta_r\)就称为Sherby-Dorn参数\(P_{SD}\)，Sherby-Dorn参数法通过建立\(\sigma- P_{SD}\)关系并利用常数激活能假设，可以基于较多的实验点计算某个应力水平、温度水平下的蠕变寿命。

小结

到这里关于单纯蠕变的基本情况差不多理清了，感觉有不少收获。不过那些真正复杂的问题还没有提及，例如变化应力下的蠕变、应力松弛等，等把疲劳整理以后再来看这些问题会更加合适。
